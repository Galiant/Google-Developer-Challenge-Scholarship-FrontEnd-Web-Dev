# Truth tables for logical AND ( && ) and logical OR ( || ).

### && (AND)
|    A	  |    B	  |  A && B  |
|:-------:|:-------:|:--------:|
|  true	  |  true	  |   true   |
|  true	  |  false	|   false  |
|  false  |  true	  |   false  |
|  false	|  false	|   false  |

### || (OR)
|    A	  |    B	  |  A || B  |
|:-------:|:-------:|:--------:|
|  true	  |  true	  |   true   |
|  true	  |  false	|   true   |
|  false	|  true	  |   true   |
|  false	|  false	|   false  |
